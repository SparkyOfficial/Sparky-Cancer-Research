# Include Directory

This directory contains C++ header files for the Tumor Digital Twin backend.

Header files will be organized by module:
- `simulation/` - Simulation engine, agents, grid
- `data/` - Data ingestion, parsers
- `evolution/` - Evolution engine, genotype tracking
- `pkpd/` - Pharmacokinetics/pharmacodynamics module
- `storage/` - Checkpoint and database management
- `grpc/` - gRPC service implementations
- `utils/` - Utility functions and helpers
